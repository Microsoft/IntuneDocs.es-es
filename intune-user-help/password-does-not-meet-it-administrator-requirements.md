# <a name="device-password-requirements-for-enrolled-devices"></a>Requisitos de contraseña de dispositivo para dispositivos inscritos

Su organización puede requerir que cree una contraseña más segura antes de permitirle el acceso a recursos profesionales o educativos. En este artículo se describen los requisitos de contraseña comunes para dispositivos Windows 10, iOS, macOS y Android. Es posible que su organización no aplique todos estos requisitos.  


Cuando una contraseña o un código de acceso ya no cumple un requisito, recibirá un mensaje de Portal de empresa. Se describirán los cambios que debe realizar. Si no se proporciona ningún detalle en el mensaje, use este artículo como referencia para compararlo con la contraseña actual.  

> [!IMPORTANT]
> Si ha cambiado la contraseña para cumplir los requisitos, pero sigue recibiendo notificaciones, reinicie el dispositivo.  

Para obtener ayuda adicional o para averiguar los requisitos específicos de su organización, póngase en contacto con el personal de soporte técnico de ti. Para conocer la información de contacto, vaya al [sitio web del Portal de empresa](https://go.microsoft.com/fwlink/?linkid=2010980).  

## <a name="windows-10-password-requirements"></a>Requisitos de contraseña de Windows 10

| Mensaje | Solución |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| La contraseña es obligatoria. | Establezca una contraseña. La organización requiere que escriba una contraseña para desbloquear el dispositivo. |
| La contraseña es demasiado sencilla. |  Asegúrese de que la contraseña no contenga números repetidos o secuenciales, como 1234 o 1111. |
| La contraseña es demasiado corta.| Actualice o establezca una contraseña con más caracteres. La organización requiere que la contraseña tenga una longitud determinada. Lo que elija realmente variará, pero la longitud mínima que pueden necesitar es de 4 caracteres y el máximo es 16. |
| La contraseña solo debe contener números. | Establezca una contraseña que solo contenga números.|
| La contraseña solo debe contener caracteres alfanuméricos. | Establezca una contraseña que contenga una combinación de números y letras.|
| La contraseña debe contener caracteres complejos. | Agregue caracteres complejos como números, letras mayúsculas y símbolos como `$`, `%`y `#`. Su organización necesita una combinación de letras, números y caracteres no alfanuméricos para que otros usuarios puedan adivinar la contraseña.|  
| La contraseña ha expirado. | Establezca una nueva contraseña. La organización requiere que cambie la contraseña después de un número determinado de días. |
| La contraseña se ha usado recientemente. | Elija una contraseña que no haya usado antes. La organización requiere que transcurra un período de tiempo determinado antes de volver a usar una contraseña. |

## <a name="ios-passcode-requirements"></a>requisitos de código de acceso de iOS

| Mensaje | Solución |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| El código de acceso es obligatorio.| Establezca un código de acceso. La organización requiere que escriba un código de acceso para desbloquear el dispositivo. |
| El código de acceso es demasiado simple. |  Asegúrese de que el código de acceso no contenga números repetidos o secuenciales, como 1234 o 1111. |
| El código de acceso es demasiado corto. | Actualizar o establecer un código de acceso con más caracteres. La organización requiere que el código de acceso tenga una longitud determinada. Lo que elija realmente variará, pero la longitud mínima que pueden necesitar es de 4 caracteres y el máximo es 14. Al cambiar el código de acceso, es posible que aparezca un mensaje de Apple que le indica que escriba 6 o más caracteres. Este mensaje es una recomendación de Apple System. Si su organización solo necesita un código de acceso de 4 o 5 caracteres, no es necesario que escriba un código de acceso de 6 dígitos.|  
| El código de acceso solo debe contener números. | Establezca un código de acceso que solo contenga números.|
| El código de acceso solo debe contener caracteres alfanuméricos.| Establecer un código de acceso que contenga una combinación de números y letras.|
| El código de acceso debe contener caracteres no alfanuméricos. | Agregue caracteres especiales como `&`, `!`, `$`, `%`y `#`. Su organización necesita una combinación de letras, números y caracteres no alfanuméricos para que otros usuarios puedan adivinar el código de acceso.|
| El código de acceso ha expirado. | Establezca una nueva contraseña. La organización requiere que cambie la contraseña después de un número determinado de días. |
| El código de acceso se usó demasiado recientemente.| Elija un código de acceso que no haya usado antes. La organización requiere que transcurra un período de tiempo determinado antes de volver a usar el código de acceso. |
|Se requiere la autenticación Touch ID o facial ID. | Configure Touch ID o facial ID. La organización requiere que se autentique con uno de estos métodos antes de usar Autorrellenar para contraseñas o información de tarjeta de crédito. |

## <a name="android-password-requirements"></a>Requisitos de contraseña de Android
| Mensaje | Solución |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| La contraseña es obligatoria. | Establezca una contraseña o un PIN. La organización requiere que escriba una contraseña para desbloquear el dispositivo. |
| La contraseña es demasiado sencilla. |  Asegúrese de que la contraseña o el PIN no contienen números repetidos o secuenciales, como 1234 o 1111. |
| La contraseña es demasiado corta. | Actualice o establezca una contraseña con más caracteres. La organización requiere que la contraseña tenga una longitud determinada.|
| La contraseña debe contener números. | Establezca una contraseña o un PIN que contenga números.|
| La contraseña debe contener letras. | Establezca una contraseña que contenga letras del alfabeto.|
| La contraseña debe contener caracteres alfanuméricos. | Establezca una contraseña que contenga una combinación de números y letras.|
| La contraseña debe contener caracteres alfanuméricos y símbolos. | Establezca una contraseña que contenga una combinación de letras, números y caracteres especiales como `&`, `!`, `$`, `%`y `#`. |
| La contraseña debe usar la tecnología biométrica.| Configure el dispositivo para que use la autenticación biométrica, como el reconocimiento de huellas digitales o facial.
| La contraseña ha expirado. | Establezca una nueva contraseña. La organización requiere que cambie la contraseña después de un número determinado de días. |
| La contraseña se ha usado recientemente. | Elija una contraseña que no haya usado antes. La organización requiere que transcurra un período de tiempo determinado antes de volver a usar una contraseña. |
