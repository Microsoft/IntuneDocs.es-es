# <a name="delivery-optimization-settings-for-intune"></a>Configuración de optimización de entrega de Intune

En este artículo se muestra la configuración de optimización de entrega compatibles con Intune para dispositivos que ejecutan Windows 10 o posterior.  

Mayoría de las opciones en la consola de Intune se asignan directamente a la configuración de optimización de entrega que se trata en profundidad en la documentación de Windows para el que se proporcionan vínculos a contenido relevante.  Configuración o las opciones específicas a Intune no contienen vínculos a contenido adicional.  

Las siguientes tablas incluyen:  

- **Establecer**: la configuración tal y como aparece en Intune. Configuración que es vínculos de abre la entrada correspondiente en [actualiza la configuración de optimización de entrega para Windows 10](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization) en la documentación de Windows donde se puede obtener más información acerca de la configuración.

- **Versión de Windows**: la versión mínima de Windows 10 que incluye compatibilidad con esta configuración.  

- **Detalles**: una breve descripción de cómo Intune implementa la configuración, incluido el valor predeterminado de Intune. Cuando esté disponible, se incluyen vínculos a [proveedor de servicios de configuración de directiva de optimización de entrega](https://docs.microsoft.com/windows/client-management/mdm/policy-csp-deliveryoptimization) entradas (CSP).  

Para configurar Intune para que use esta configuración, consulte [entregar actualizaciones](delivery-optimization-windows.md).
